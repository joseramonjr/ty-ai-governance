## [PRE-FILLED -- from core doctrine]

my TY AI OS installation if ALL of the following
are simultaneously true:

  1. My installation has been compromised in
     a way that cannot be remediated within

     harm to humans

     can be reached to authorize remediation

     corrupted beyond recovery

     attack -- freeze instead

     guardian through the closed chain

     in the append-only ledger

     halt condition detection before executing
